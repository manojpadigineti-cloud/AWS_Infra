[1] End-to-end CI/CD workflow? Pipeline type?
Git push/PR → webhook triggers Jenkins Multibranch Declarative Pipeline → parallel build/test/scan → artifact Nexus → ArgoCD sync K8s → canary → promote → blue-green prod. Declarative: readable, stage visualization, error handling.

[2] Declarative vs Scripted pipelines?
Declarative: pipeline { agent any; stages { stage('Build') {} } }. Structured, BlueOcean friendly, post{} blocks. Scripted: node { stage('Build') { sh 'make' } }. Flexible but harder maintain. Use Declarative 95% cases.

[3] Define/trigger Jenkins pipelines?
Jenkinsfile in repo root. Multibranch Pipeline scans branches/PRs. Triggers: GitHub webhook, pollSCM H/5 * * * *, Upstream job. Parameters override via API/UI.

[4] Jenkins Shared Libraries?
@Library('shared-lib@master') _ . vars/deploy.groovy: def call(Map config) { ... }. Reusable steps across repos. Versioned branches/tags. Global Pipeline Libraries in Jenkins config.

[5] Docker works locally fails Jenkins?
Missing Docker daemon: agent { docker { image 'node:18' } } or docker.insecureHost. User perms: docker group. Volume mounts /var/run/docker.sock. Network: --network host.

[6] CI/CD pipeline?
Continuous Integration: frequent merges automated tests. Continuous Delivery/Deployment: automated releases. Stages: source→build→test→package→deploy→monitor. GitOps end-state.

[7] Connect Jenkins to EC2?
EC2 Plugin: cloud config AMI, instance cap, labels (build/node). Spot instances cost-save. JNLP port 50000 security group. IAM role JenkinsExecutionPolicy.

[8] Jenkins authenticate AWS?
IAM Role EC2 preferred: withAWS(role:'JenkinsRole', region:'us-east-1'). No keys rotation. Test: sh 'aws sts get-caller-identity'.

[9] IAM user vs role?
Role: temporary creds, no rotation, principal-based. User: long-lived keys risky. Role for Jenkins EC2/ECS agents. Policy: ec2:RunInstances*, eks:*, ecr:*.

[10] Store secrets Jenkins?
Credentials Store → ID 'aws-secret'. withAWS(credentials:'aws-secret') or withCredentials([aws(credentialsId:'aws-secret')]). Masked logs. Vault integration dynamic.

[11] Pipeline fails resource creation?
Terraform: terraform apply -auto-approve → catchError(buildResult:'UNSTABLE'). Rollback: terraform destroy -auto-approve. Slack notify. Archive tfplan/logs.

[12] Troubleshoot Jenkins failures?
Jenkins logs /var/log/jenkins/jenkins.log. BlueOcean UI. /job/lastBuild/console. Agent logs /computer/agent/log. Pipeline:UnitTest. Metrics Prometheus.
