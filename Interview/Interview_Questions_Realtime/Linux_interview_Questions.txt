[1] What is sed -E option? (Detailed explanation)
sed stream editor processes text streams line-by-line substitution/delete/insert. -E extended regex (ERE) |() {}?+ non-escaped unlike basic BRE. sed -E 's/([0-9]+)-([0-9]+)/\2-\1/g' swap dates vs sed -r legacy. Production: sed -i inplace edit config files CI/CD.

[2] Check DNS entry system?
nslookup domain.com OR dig domain.com @8.8.8.8 +short. host domain.com. /etc/resolv.conf nameserver. netstat -rn routing. Troubleshoot: dig +trace authoritative servers.

[3] Environment variables?
Key-value pairs shell process env export APP_PORT=8080 child inherits. Persistent /etc/environment ~/. rc systemd Environment=. Read $VAR ${VAR:-default}. CI/CD secrets mask.

[4] Cron job create?

[7] Systemd logs missing reboot?

[8] Kill process Ubuntu?

[9] Open firewall?
ufw allow 8080/tcp sudo ufw reload status. iptables -A INPUT -p tcp --dport 8080 -j ACCEPT. firewalld firewall-cmd --add-port=8080/tcp permanent reload.

[10] Check open ports?
netstat -tlnp OR ss -tlnp listen TCP/UDP PID. lsof -i :8080. nmap localhost.

[11] netstat -ntlp?
-n numeric IP/port no resolve -t TCP -l listen -p PID/program. ss -tlnp modern replacement.

[13] Clear cache?
echo 3 > /proc/sys/vm/drop_caches (pages dentries inodes) prod careful. systemd-tmpfiles --clean. free -h before/after.

[14] Forward Proxy?
Client → proxy → internet Squid/V2Ray caches/auth hides client IP. Browser PAC config http_proxy=proxy:3128.

[15] Reverse Proxy?
Internet → proxy → backend servers nginx/HAProxy L7 LB TLS termination. Hide internal IPs.

[16] Forward vs Reverse Proxy?
Forward client-side internet access corp firewall. Reverse server-side LB SSL offload WAF. Same nginx config different use.
