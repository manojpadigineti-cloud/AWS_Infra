[3] Automate log cleanup?
Used Python scripts scheduled via cron to delete logs older than retention policy (e.g., 7 days) to prevent disk issues.
Example:

 python
import os, time

log_dir = "/var/log/myapp"
days = 7
now = time.time()

for file in os.listdir(log_dir):
    path = os.path.join(log_dir, file)
    if os.stat(path).st_mtime < now - days*86400:
        os.remove(path)
 

[4] Check SSL expiry?
Used ssl and socket modules to monitor certificate expiry and alert before expiration.
Example:

 python
import ssl, socket
from datetime import datetime

hostname = "google.com"
ctx = ssl.create_default_context()

with socket.create_connection((hostname, 443)) as sock:
    with ctx.wrap_socket(sock, server_hostname=hostname) as ssock:
        cert = ssock.getpeercert()
        expiry = datetime.strptime(cert['notAfter'], '%b %d %H:%M:%S %Y %Z')
        print("Expiry:", expiry)
