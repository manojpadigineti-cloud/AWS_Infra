SSL Tunneling for EKS cluster:
===============================
Describe custer to get cluster endpoint 
----------------------------------------
aws eks describe-cluster   --name eks-private   --region us-east-1   --query "cluster.endpoint"   --output text

Edit Host File:
---------------

---------------------------------
ssh -i mkey.pem `
-L 6443:AAFBBAD7EF4B0123147E37B2F860B2DA.gr7.us-east-1.eks.amazonaws.com:443 `
ec2-user@44.197.242.57 `
-N
